This is our submission for the BPhO (British Physics Olympiad) Computational Challenge 2024 (Modelling Projectiles).
By Mantas Robinson and Bert Chun Chang.

TO RUN THE APPLICATION:
Download all of the files in the "final" directory.
Run "main.py".

NOTE THAT:
The ui-files are for reference only (made from QtDesigner), but the GUI in the finished application has a few changes from these files.
